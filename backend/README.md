# Backend

FastAPI backend service for Kalitka.

## Scope

This directory is reserved for the backend application structure, configuration, tests, and service packaging.

Do not add registration, payment, VPN, or public API business logic until the backend architecture is defined.

## Planned Structure

- `app/` - FastAPI application package
- `tests/` - backend test suite
- `Dockerfile` - backend container definition
- `pyproject.toml` - Python project metadata and dependencies
