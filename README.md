# Kalitka

Kalitka is a commercial VPN service under development.

This repository currently contains only the project foundation:

- `backend/` - FastAPI service skeleton
- `frontend/` - Next.js application skeleton
- `docs/` - project documentation
- `infra/` - infrastructure configuration for Docker Compose, PostgreSQL, and Nginx
- `scripts/` - local development and operations scripts

No registration, payments, API endpoints, VPN logic, or frontend screens are implemented yet.

## Getting Started

The project is intentionally minimal at this stage. Read the README files in each directory before adding implementation code.
